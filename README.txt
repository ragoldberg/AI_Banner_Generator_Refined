
AI Banner Generator - Refined
=============================

This tool allows you to generate banners with AI-generated text and styles.

Instructions:
1. Install Python 3.x on your computer.
2. Install required libraries: `pip install pillow tkinter`.
3. Run the script: `python AI_Banner_Generator.py`.
4. Input a background style and text prompt to generate a banner.
5. Preview and save the banner.

Enjoy!
